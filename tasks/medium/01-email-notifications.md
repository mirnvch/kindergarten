# Email-уведомления

## Описание
Настроить отправку email-уведомлений через Resend API (уже сконфигурирован).

## Требования

### Типы уведомлений
1. **Аутентификация**
   - [ ] Подтверждение регистрации
   - [ ] Сброс пароля
   - [ ] Смена email

2. **Бронирования**
   - [ ] Новое бронирование (для владельца)
   - [ ] Подтверждение бронирования (для родителя)
   - [ ] Отмена бронирования
   - [ ] Напоминание за 24 часа

3. **Сообщения**
   - [ ] Новое сообщение (если не прочитано за 1 час)

4. **Отзывы**
   - [ ] Новый отзыв (для владельца)
   - [ ] Ответ на отзыв (для родителя)
