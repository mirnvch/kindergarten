# Система списка ожидания

## Описание
Реализовать функционал записи в очередь для детских садов без свободных мест.

## Требования

### Для родителей
- [ ] Кнопка "Записаться в очередь" (если нет мест)
- [ ] Просмотр позиции в очереди
- [ ] Отмена записи
- [ ] Уведомление при появлении места

### Для владельцев
- [ ] Просмотр списка ожидания
- [ ] Приглашение из очереди
- [ ] Удаление из очереди
- [ ] Настройка лимита мест

### Логика
1. При записи - добавление в конец очереди
2. При появлении места - уведомление первого в очереди
3. Срок для ответа - 48 часов
4. При истечении - переход к следующему

### Database
Модель `WaitlistEntry` уже существует в schema.prisma

## Критерии готовности
- Родитель может записаться в очередь
- Родитель видит позицию в очереди
- Владелец может управлять очередью
- Автоматические уведомления работают
