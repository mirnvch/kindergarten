# Мобильная адаптация

## Описание
Улучшить мобильную версию сайта.

## Требования

### Общие улучшения
- [ ] Проверить все страницы на мобильных устройствах
- [ ] Исправить проблемы с версткой
- [ ] Улучшить touch-friendly элементы
- [ ] Оптимизировать размеры кнопок (min 44px)

### Навигация
- [ ] Мобильное меню (hamburger)
- [ ] Bottom navigation bar
- [ ] Swipe gestures где уместно

### Формы
- [ ] Правильные типы input (tel, email)
- [ ] Оптимизация клавиатуры
- [ ] Автозаполнение

### Изображения
- [ ] Responsive images
- [ ] Lazy loading
- [ ] Правильные размеры для мобильных

### Компоненты для проверки
- [ ] Header/Navigation
- [ ] Search filters
- [ ] Daycare cards
- [ ] Booking forms
- [ ] Messages UI
- [ ] Portal dashboard

### Тестирование
- iPhone SE (375px)
- iPhone 12/13 (390px)
- Android (360px)
- Tablet (768px)

## Критерии готовности
- Все страницы корректно отображаются на мобильных
- Формы удобны для заполнения
- Навигация интуитивна
- Нет горизонтального скролла
