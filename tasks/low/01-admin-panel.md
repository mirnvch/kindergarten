# Админ-панель

## Описание
Создать панель администратора для модерации контента и управления платформой.

## Требования

### Страницы
- `/admin/dashboard` - обзор метрик
- `/admin/users` - управление пользователями
- `/admin/daycares` - модерация детских садов
- `/admin/reviews` - модерация отзывов
- `/admin/reports` - жалобы пользователей
- `/admin/settings` - настройки платформы

### Dashboard метрики
- [ ] Общее количество пользователей
- [ ] Количество детских садов
- [ ] Количество бронирований за период
- [ ] Выручка платформы
- [ ] График роста

### Управление пользователями
- [ ] Список с поиском и фильтрами
- [ ] Просмотр профиля
- [ ] Блокировка/разблокировка
- [ ] Смена роли
- [ ] Удаление аккаунта

### Модерация детских садов
- [ ] Верификация новых садов
- [ ] Приостановка листинга
- [ ] Редактирование контента

### Модерация отзывов
- [ ] Список жалоб
- [ ] Удаление неуместных отзывов
- [ ] Предупреждения пользователям

### Server Actions
```typescript
// src/server/actions/admin.ts
getAdminStats()
getUsers()
updateUser()
blockUser()
getDaycares()
verifyDaycare()
suspendDaycare()
getReports()
resolveReport()
```

## Критерии готовности
- Админ может просматривать статистику
- Админ может управлять пользователями
- Работает модерация контента
- Логирование действий админа (AuditLog)
