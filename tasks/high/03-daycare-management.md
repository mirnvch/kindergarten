# Управление детским садом (Portal)

## Описание
Расширить функционал портала для владельцев детских садов.

## Требования

### Страницы портала
- `/portal/daycare/edit` - редактирование профиля сада
- `/portal/bookings` - управление бронированиями
- `/portal/enrollments` - управление зачислениями
- `/portal/schedule` - настройка расписания
- `/portal/pricing` - управление ценами
- `/portal/photos` - загрузка фотографий

### Редактирование профиля
- [ ] Основная информация (название, описание)
- [ ] Адрес и координаты
- [ ] Контактные данные
- [ ] Выбор удобств (amenities)
- [ ] Программы обучения
- [ ] Возрастные группы

### Управление бронированиями
- [ ] Список всех бронирований с фильтрами
- [ ] Подтверждение/отклонение заявок
- [ ] Календарь занятости
- [ ] Отметка NO_SHOW

### Управление расписанием
- [ ] Настройка рабочих часов по дням
- [ ] Настройка доступных слотов для экскурсий
- [ ] Праздничные/выходные дни

## Server Actions
```typescript
// src/server/actions/portal.ts
updateDaycareProfile()
updateDaycareSchedule()
updateDaycarePricing()
updateDaycareAmenities()
manageBooking()
```

## Критерии готовности
- Владелец может полностью редактировать профиль сада
- Владелец может управлять всеми бронированиями
- Работает загрузка и удаление фотографий
