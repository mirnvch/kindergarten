# Полный flow бронирования

## Описание
Реализовать полный процесс бронирования экскурсии или зачисления в детский сад.

## Требования

### Типы бронирований
1. **TOUR** - экскурсия/знакомство с садом
2. **ENROLLMENT** - заявка на зачисление

### Flow для экскурсии
1. Выбор даты и времени из доступных слотов
2. Выбор ребёнка (если несколько)
3. Добавление комментария/вопросов
4. Подтверждение бронирования
5. Email-уведомление обеим сторонам

### Flow для зачисления
1. Заполнение формы заявки
2. Выбор программы и расписания
3. Загрузка документов (опционально)
4. Оплата регистрационного сбора (если есть)
5. Перевод в статус PENDING

### Страницы
- `/daycare/[slug]/book` - форма бронирования
- `/daycare/[slug]/enroll` - форма зачисления
- `/booking/[id]/confirmation` - подтверждение

### Server Actions
```typescript
// src/server/actions/booking.ts
createTourBooking()
createEnrollmentRequest()
cancelBooking()
rescheduleBooking()
confirmBooking() // для владельца
```

## Критерии готовности
- Родитель может забронировать экскурсию
- Родитель может подать заявку на зачисление
- Владелец получает уведомление о новой заявке
- Работает отмена и перенос бронирования
