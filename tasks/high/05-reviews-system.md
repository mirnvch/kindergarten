# Система отзывов и рейтингов

## Описание
Реализовать возможность оставлять и просматривать отзывы о детских садах.

## Требования

### Для родителей
- [ ] Форма создания отзыва
- [ ] Общий рейтинг (1-5 звёзд)
- [ ] Детальные оценки:
  - Безопасность (safety)
  - Чистота (cleanliness)
  - Персонал (staff)
  - Программы (programs)
  - Соотношение цена/качество (value)
- [ ] Текстовый отзыв
- [ ] Редактирование своего отзыва

### Для владельцев
- [ ] Просмотр всех отзывов
- [ ] Ответ на отзыв
- [ ] Жалоба на неуместный отзыв

### Отображение
- [ ] Список отзывов на странице сада
- [ ] Средний рейтинг и количество отзывов
- [ ] Сортировка (новые, полезные)
- [ ] Фильтр по рейтингу

### Валидация
- Только пользователи с завершённым бронированием могут оставить отзыв
- Один отзыв на пользователя на сад

### Server Actions
```typescript
// src/server/actions/reviews.ts
createReview()
updateReview()
deleteReview()
replyToReview()
getReviews()
reportReview()
```

## Критерии готовности
- Родитель может оставить детальный отзыв
- Отзывы отображаются на странице сада
- Владелец может отвечать на отзывы
- Рейтинг автоматически пересчитывается
