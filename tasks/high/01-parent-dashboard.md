# Dashboard для родителей

## Описание
Создать полноценный личный кабинет для родителей с возможностью управления профилем, детьми и бронированиями.

## Требования

### Основные страницы
- `/parent/dashboard` - главная страница с обзором
- `/parent/children` - управление профилями детей
- `/parent/bookings` - история и активные бронирования
- `/parent/favorites` - избранные детские сады
- `/parent/messages` - входящие сообщения

### Функциональность
- [ ] Виджет с предстоящими бронированиями
- [ ] Статус заявок на зачисление
- [ ] Быстрый доступ к избранным садам
- [ ] Уведомления о новых сообщениях
- [ ] CRUD операции для профилей детей

### Компоненты
- `ParentDashboardStats` - статистика
- `UpcomingBookings` - предстоящие визиты
- `ChildrenList` - список детей
- `FavoritesDaycares` - избранные сады

## Технические детали
- Использовать существующие Server Actions
- Добавить новые actions в `/src/server/actions/parent.ts`
- Защита маршрутов через middleware (role: PARENT)

## Критерии готовности
- Родитель может видеть все свои бронирования
- Родитель может добавлять/редактировать детей
- Родитель может управлять избранными садами
